# D-Secure Product Catalog & Technical Details

This document provides deep technical insight into D-Secure's software solutions for data erasure and hardware diagnostics.

## 1. D-Secure Drive Eraser (Standard & +Diagnostics)
Professional-grade data sanitization for all types of storage media.

### Core Capabilities
- **Mass Erasure:** Wipe up to 64 drives simultaneously on a single machine.
- **Bootable Environment:** Runs in its own secure environment, independent of the host OS (Linux-based kernel).
- **Standards:** Supports NIST 800-88 Purge & Clear, DoD 5220.22-M, and 20+ other algorithms.
- **Hardware Support:** HDD, SSD (SATA, NVMe, SAS), Mac (Intel & Apple Silicon), Chromebooks, and Servers.

### Drive Eraser + Diagnostics (Variant)
Includes all features of the standard version plus:
- **Integrated Health Check:** Run SMART tests before erasure.
- **Hardware Grading:** Automatically grade devices (A, B, C) based on health and physical condition.
- **SMART Health Analysis:** Detailed reporting on drive life, temperature, and sector health.

---

## 2. D-Secure Hardware Diagnostics
Comprehensive hardware health testing for ITADs and refurbishers.

### Testing Scope
- **CPU & GPU:** Stress tests and benchmark validation.
- **RAM:** Full bit-level memory testing.
- **Motherboard:** Chipset and bus integrity checks.
- **Battery:** Cycle count, health percentage, and discharge rate analysis.
- **Peripherals:** Keyboard, Screen (dead pixels), Webcam, and Audio tests.
- **Network:** Wi-Fi and Ethernet connectivity benchmarks.

---

## 3. D-Secure Smart Diagnostic
A specialized tool for proactive hard drive monitoring and cloning.

### Key Features
- **Real-time Monitoring:** Alerts users before a drive fails.
- **Bad Sector Scanning:** Identifies damaged areas on the disk.
- **Sector-by-Sector Cloning:** Safely clone data from a dying drive to a new one.
- **Health Dashboard:** Visual representation of drive longevity and temperature.

---

## 4. D-Secure File Eraser Professional
Targeted data sanitization for active Windows environments.

### Use Cases
- **Specific File/Folder Wiping:** Permanently delete sensitive documents without wiping the whole drive.
- **Free Space Cleaning:** Wipe "slack space" where deleted files might still exist.
- **Application Traces:** Clean browser history, system logs, and temporary files securely.
- **Volume Erasure:** Wipe entire partitions or logical drives.
